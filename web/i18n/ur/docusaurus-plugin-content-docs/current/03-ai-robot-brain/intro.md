# ماڈیول 3: AI Robot Brain

## تعارف

ماڈیول 3 میں خوش آمدید، جہاں آپ جدید روبوٹ کے "دماغ" کے طور پر مصنوعی ذہانت اور machine learning میں غوطہ لگائیں گے۔

## آپ کیا سیکھیں گے

اس ماڈیول میں، آپ:
- روبوٹکس کے تناظر میں AI کی بنیادی باتیں سمجھیں گے
- neural networks اور deep learning کے بارے میں سیکھیں گے
- روبوٹ کے لیے فیصلہ سازی algorithms کو دریافت کریں گے
- روبوٹک ایپلیکیشنز کے لیے بنیادی AI ماڈلز کو implement کریں گے

## پیشگی ضروریات

- ماڈیول 1 اور 2 کی تکمیل
- Python پروگرامنگ (درمیانی سطح)
- بنیادی ریاضیات (الجبرا اور شماریات)

## شروع کریں

دریافت کریں کہ کیسے AI روبوٹ کو programmed مشینوں سے ذہین سسٹمز میں تبدیل کرتا ہے جو سیکھنے اور ڈھالنے کے قابل ہیں۔
