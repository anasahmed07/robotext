# ماڈیول 1: Robotic Nervous System

## تعارف

ماڈیول 1 میں خوش آمدید، جہاں آپ روبوٹک سسٹمز کی بنیادی ساخت - Robotic Nervous System (RNS) کے بارے میں سیکھیں گے۔

## آپ کیا سیکھیں گے

اس ماڈیول میں، آپ:
- Robotic Nervous System کے تصور کو سمجھیں گے
- سیکھیں گے کہ روبوٹ کس طرح sensory معلومات کو محسوس اور process کرتے ہیں
- روبوٹک سسٹمز کے اندر مواصلاتی راستوں کو دریافت کریں گے
- جدید روبوٹکس موضوعات کے لیے بنیادی علم بنائیں گے

## پیشگی ضروریات

- کمپیوٹر سسٹمز کی بنیادی سمجھ
- Python پروگرامنگ سے واقفیت (ابتدائی سطح)
- روبوٹ کیسے کام کرتے ہیں اس میں تجسس!

## شروع کریں

شروع کرنے کے لیے تیار ہیں؟ آئیے سمجھنے سے شروع کریں کہ روبوٹ اپنے ماحول سے "آگاہ" کیسے ہوتے ہیں۔
